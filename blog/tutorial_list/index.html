Tutorials

These tutorials utilize the Docker system which avoid the tedius process of installing the many programs needed to follow tutorials. You can continue learning using the Docker images used in these tutorials and continue to do your own original research with them. Or you can install the programs used in the tutorial yourself by inspecting the Dockerfile containers matching the tutorial. A pre-requisite for all of these tutorials is thus the Docker program. You may install Docker within your OS package manager or try here.
